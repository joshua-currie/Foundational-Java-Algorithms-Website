Fibonacci Recursive

Description

Given an integer representing which term of the Fibonacci Sequence to compute, will compute that term of Fibonacci using brute-force unoptimized recursion.

Input Requirements

Takes in one command line argument that is an integer representing which term of Fibonacci to compute.


Example Input and Output

Input: 7

Output: The 7term of Fibonacci using brute force recursion is 13


Time and Space Complexity

Worse case time: O(2n)

Best case time: O(2n)

Average case time: O(2n)

Worse case space: O(n)

Best case space: O(n)

Average case space: O(n)
