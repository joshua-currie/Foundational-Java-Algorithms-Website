Sorting Algorithms

This page contains links to six sorting algorithms I coded in Java as well

as a Java program that contains all six algorithms into one

Bubble Sort

Description

Given an array of integers, will sort the array by iterating through the given array repeatedly and swap adjacent elements if they are in the wrong order.

After every iteration, the size of the array looked at by bubble sort is decreased by one element as each iteration gurentees that one additional element located towards the end of array is in the proper position.

Bucket Sort

Description

Given an array of integers, will sort the array by first finding the largest integer and then create a separate integer array that is initialized to 0 with size of the largest integer.

Then loop through each integer in the given array and increment the integer in the created array at the index that represents it.

Finally, iterate through the created array from smallest index to largest index, checking if that index contains a number that is not 0. If so, then add the number with represented by that index on to our sorted list array and decrement the number at the index.

Insertion Sort

Description

Given an array of integers, will sort the array by iterating though the array and building up to the final sorted version of the array one element per iteration.

Insertion sort first looks at the second element in the array and compares its position to the first element. If they are not in the correct position then swap the two elements. At this point, the first two elements of the array are sorted.

Insertion sort then looks at the next element in the array and places its position in the already sorted portion of the array, incrementing the size of the sorted section of the array by one. This process repeats until the entire array is sorted.

Selection Sort

Description

Given an array of integers, will sort the array by iterating though the array and building up to the final sorted version of the array one element per iteration.

Selection sort first iterates through the entire array to find the smallest array element and then swaps that element with the element located at index 0. Now the element at index 0 is in the right place.

Then it iterates through the entire array again to find the smallest array element but ignories the element at index 0. After finding it, swaps the element with the element at index 1.

This process of building up the sorted array from finding the smallest element to the largest element is repeated until the array is fully sorted.

Merge Sort

Description

Given an array of integers, will sort the array by dividing the array into smaller arrays, sorting those arrays individually, and recombining the sorted arrays to form the complete sorted array.

Merge sort first recursively divides the given array into smaller halves until all the sub-arrays are of size 1. Afterwards, the sub-arrays are then merged together to sub-arrays of size 2 by comparing the first (and currently only) elements in the subarrays and merging them together in sorted order.

This process of looking and comparing the first elements of the sub-arrays in question to merge the two sub-arrays into one array is repeated until all the sub-arrays are finally merged back into one array.

Quicksort

Description

Given an array of integers, will sort the array by first selecting an element to be the pivot. Then the array is partitioned into two sub-arrays, where one sub-array contains only the elements that are smaller than the pivot element, and the other sub-array contains only the elements that are larger than the pivot element.

This process is then recursively repeated in the sub-arrays by selecting a pivot in each sub-array and dividing each sub-array into more sub-arrays until the sub-arrays are of size one.

The sub-arrays are then combined back together until there is only one array.

All Sorting Algorithms

Description

Given an array of integers and an integer representing which sorting algorithm to perform, will sort the array using the selected method.

The options for sorting algothims are: Bubble sort, Bucket sort, Insertion sort, Selection sort, Merge sort, and Quicksort.
