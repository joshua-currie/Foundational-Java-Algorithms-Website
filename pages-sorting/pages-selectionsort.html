Selection Sort

Description

Given an array of integers, will sort the array by iterating though the array and building up to the final sorted version of the array one element per iteration.

Selection sort first iterates through the entire array to find the smallest array element and then swaps that element with the element located at index 0. Now the element at index 0 is in the right place.

Then it iterates through the entire array again to find the smallest array element but ignories the element at index 0. After finding it, swaps the element with the element at index 1.

This process of building up the sorted array from finding the smallest element to the largest element is repeated until the array is fully sorted.


Input Requirements

Takes in one command line argument that is a String of integers where each integer is separated by a comma and no spaces.


Example Input and Output

Input: 6,5,3,6,8,1,4,3

Output: Array after selection sort: 1 3 3 4 5 6 6 8


Time and Space Complexity

Worse case time: O(n2)

Best case time: O(n)

Average case time: O(n2)

Worse case space: O(n)

Best case space: O(n)

Average case space: O(n)
