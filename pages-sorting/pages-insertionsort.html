Insertion Sort

Description

Given an array of integers, will sort the array by iterating though the array and building up to the final sorted version of the array one element per iteration.

Insertion sort first looks at the second element in the array and compares its position to the first element. If they are not in the correct position then swap the two elements. At this point, the first two elements of the array are sorted.

Insertion sort then looks at the next element in the array and places its position in the already sorted portion of the array, incrementing the size of the sorted section of the array by one. This process repeats until the entire array is sorted.


Input Requirements

Takes in one command line argument that is a String of integers where each integer is separated by a comma and no spaces.


Example Input and Output

Input: 6,5,3,6,8,1,4,3

Output: Array after insertion sort: 1 3 3 4 5 6 6 8


Time and Space Complexity

Worse case time: O(n2)

Best case time: O(n)

Average case time: O(n2)

Worse case space: O(n)

Best case space: O(n)

Average case space: O(n)
