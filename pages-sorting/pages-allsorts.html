All Sorting Algorithms

Description

Given an array of integers and an integer representing which sorting algorithm to perform, will sort the array using the selected method.

The options for sorting algothims are: bubble sort (1), bucket sort (2), insertion sort (3), selection sort (4), merge sort (5), and quicksort (6).


Input Requirements

Takes in two command line arguments. The first one being a number from 1-6 that signifies which sorting algorithm to use.

The second command line argument takes in a String of integers where each integer is separated by a comma and no spaces.


Example Input and Output

Input: 4 6,5,3,6,8,1,4,3

Output: Array after selection sort: 1 3 3 4 5 6 6 8
